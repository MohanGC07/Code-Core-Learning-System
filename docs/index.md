## 🧭 How to Use This Platform

!!! tip "New here? Follow these steps"

    1. **Pick a track** from the cards above
    2. **Open Module 01** and start at Lesson 01
    3. **Read actively** — open your editor alongside the lesson
    4. **Do every exercise** — typing beats reading every time
    5. **Try the challenge** at the end of each lesson

!!! success "What makes this different"
    Every lesson follows the same proven structure — concept,
    analogy, code example, exercise, challenge, and quiz.
    You always know what to expect and what to do next.
